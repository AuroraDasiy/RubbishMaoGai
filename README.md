# RubbishMaoGai
Auto generate database of questions based on known results for http://course.xmu.edu.cn/  
自动根据已做结果生成厦大course毛概课的题库  

`samples`: 原始测试网页  
`tiku.py`: 生成题库的脚本  

## 如何提供题目?
### 1 完成6次测试
### 2 点击查看结果，进入含有答案的结果页面
### 3 右键分别保存6个测试结果网页
### 4 将网页结果打包发送至作者
### 5 或者放入samples文件夹内，并发起pull request
